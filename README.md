
#  Koshiv Logistics Website

![Website](https://img.shields.io/badge/website-up-brightgreen)

This repository contains the source code for the official website of Koshiv Logistics, a modern tech-driven logistics company launching in 2026.

## 📜 About

Koshiv Logistics is a new-age logistics and trucking company based in Mumbai, India. We aim to revolutionize the import/export transportation industry by leveraging technology to provide seamless, efficient, and transparent services. This website serves as the online presence for the company, providing information about our services, fleet, and vision.

## ✨ Features

*   **Services Information:** Detailed information about the services we offer, including cross-border transportation, customs clearance, and specialized cargo handling.
*   **Fleet Showcase:** A comprehensive look at our modern and diverse fleet of trucks and trailers.
*   **Company Story:** Learn more about our mission, vision, and the team behind Koshiv Logistics.
*   **Contact Form:** An easy way for potential clients and partners to get in touch with us.
*   **Responsive Design:** The website is designed to be accessible and user-friendly across all devices.

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3, JavaScript
*   **Frameworks/Libraries:** Google Fonts, Font Awesome
*   **Deployment:** GitHub Pages

## 🚀 How to View

To view the website, you can:
1.  Visit the live website [here](https://koushaljhacs.github.io/demo/).
2.  Clone this repository and open the `index.html` file in your browser.

```bash
git clone https://github.com/koushaljhacs/KOSHIV.git
cd KOSHIV
# Open index.html in your browser
```
